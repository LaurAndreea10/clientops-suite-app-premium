# ClientOps Suite

A zero-cost SaaS front-end prototype built with Vite, React, TypeScript, Zustand, Recharts, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

This starter uses `HashRouter` and `base: './'`, which keeps it safe for GitHub Pages deployments without server routing.
